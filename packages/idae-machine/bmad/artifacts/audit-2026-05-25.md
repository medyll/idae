   Contains `console.warn(...)` for missing collections during seeding. Should use the logger utility or silent fallback.

3. **Minor — `src/lib/main/machine/MachineError.ts`**
   Contains `console.error(...)` in error handlers. Should delegate to logger utility.

4. **Minor — `src/lib/main/machine/MachineSchemeValues.ts`**
   Contains `// TODO: Refine IDbForge type if needed for stricter typing`. **Minor** — one outstanding TODO.

5. **Minor — Dead CSS in `Explorer.svelte`**
   `.explorer-toolbar` and `.mode-switcher` selectors have no matching markup. Remove the blocks.

6. **Minor — Prettier drift**
   `pnpm run lint` still reports formatting warnings across many files (mostly artifacts and docs). Recommend running `pnpm run format` and committing.
