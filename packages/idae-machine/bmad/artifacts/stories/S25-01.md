# S25-01 — Refactor AppShell.svelte — snippets navbar/sidebar/content

**Sprint:** 25
**Effort:** M
**Priority:** high
**Dependencies:** S24-04 (Frame.svelte), S24-05 (machine.loadFrame)

---

## Context

`AppShell.svelte` existe mais est orphelin — la logique est inline dans `+layout.svelte`.
L'objectif S25 est de faire d'AppShell le template universel pour tous les frames de l'app.

## Goal

Réécrire AppShell pour accepter trois snippets nommés (`navbar`, `sidebar`, `content`) avec des defaults opérationnels:
- **navbar default**: titre app + toggle DEV panel
- **sidebar default**: `DataList[appscheme]` — liste réactive des collections, click → `machine.loadFrame('explorer', code, undefined, { mode: 'list' })`
- **content default**: `<Frame id={instanceName} />`

## Files

| File | Change |
|------|--------|
| `src/lib/shell/layout/AppShell.svelte` | Réécriture complète |

## Acceptance

- Props: `instanceName?: string` (default `'main'`)
- `{#snippet navbar()}`, `{#snippet sidebar()}`, `{#snippet content()}` — tous overridables
- Sidebar: `DataList collection="appscheme" sortBy={{ field: 'order', direction: 'asc' }}` + loop `{#each items as row}` → button onclick `machine.loadFrame('explorer', row.code, undefined, { mode: 'list' })`
- activeCollection highlight: comparé à `machineFrameManager` current collection state
- DevResetPanel embarqué dans navbar (DEV only)
- Svelte 5 runes uniquement (`$props`, `$state`, `$derived`)
- 0 import de PaneLeft, PaneCollectionGroup

## Spec interface

```svelte
<script lang="ts">
  let {
    instanceName = 'main',
    navbar,
    sidebar,
    content,
  }: {
    instanceName?: string;
    navbar?:   Snippet;
    sidebar?:  Snippet;
    content?:  Snippet;
  } = $props();
</script>

<div class="app-shell">
  <header>{#if navbar}{@render navbar()}{:else}<!-- default navbar -->{/if}</header>
  <div class="app-body">
    <aside>{#if sidebar}{@render sidebar()}{:else}<!-- default DataList[appscheme] -->{/if}</aside>
    <main>{#if content}{@render content()}{:else}<Frame id={instanceName} />{/if}</main>
  </div>
</div>
```
