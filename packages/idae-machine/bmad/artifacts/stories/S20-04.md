## Context

Legacy avait module images via GridFS + crops à la volée. idae-machine n'a rien. Goal : extension de `FileService` qui détecte les uploads image et :
- extrait dimensions + metadata basiques
- génère thumbnails à tailles standard à la demande (lazy, cached)
- sert variants via endpoint dédié

Pas de manipulation destructive sur l'original. Variants cachés sur disque.

## Variants standard

| Variant | Largeur max | Hauteur max | Mode |
|---------|-------------|-------------|------|
| `thumb` | 150 | 150 | cover (crop center) |
| `small` | 480 | — | inside (fit) |
| `medium` | 1024 | — | inside |
| `large` | 2048 | — | inside |

Toujours JPEG output qualité 82 (sauf si original PNG avec alpha → PNG output). WebP optionnel via `?fmt=webp`.

## Flow

```
POST /api/files/:coll/:id   (multipart, image/jpeg)
  └── FileService.uploadFile
        └── if (mimetype.startsWith('image/')) → ImageService.probe(storedPath) → patch metadata.image
```

```
GET /api/files/:fileId/image/:variant
  └── FileService.getFileMeta(fileId)
  └── ImageService.getVariant(storedPath, variant)
        ├── compute cache path: storedPath + `.${variant}.jpg`
        ├── if exists → stream
        └── else      → sharp resize → write cache → stream
```

## ImageService

```ts
import sharp from 'sharp';
import fs from 'fs';
import path from 'path';
import type { ImageMeta, ImageVariant } from '../types/imageMeta.js';

export async function probe(filePath: string): Promise<ImageMeta | null> {
	try {
		const meta = await sharp(filePath).metadata();
		return {
			width:    meta.width ?? 0,
			height:   meta.height ?? 0,
			format:   meta.format ?? 'unknown',
			hasAlpha: !!meta.hasAlpha,
		};
	} catch { return null; }
}

const VARIANTS: Record<ImageVariant, { w: number; h?: number; fit: 'cover' | 'inside' }> = {
	thumb:  { w: 150,  h: 150, fit: 'cover' },
	small:  { w: 480,         fit: 'inside' },
	medium: { w: 1024,        fit: 'inside' },
	large:  { w: 2048,        fit: 'inside' },
};

export async function getVariant(
	storedPath: string,
	variant: ImageVariant,
	fmt: 'jpeg' | 'webp' = 'jpeg'
): Promise<string> {
	const cfg = VARIANTS[variant];
	if (!cfg) throw new Error(`Unknown variant: ${variant}`);
	const ext = fmt === 'webp' ? 'webp' : 'jpg';
	const cachePath = `${storedPath}.${variant}.${ext}`;
	if (fs.existsSync(cachePath)) return cachePath;

	let pipeline = sharp(storedPath).resize({ width: cfg.w, height: cfg.h, fit: cfg.fit, withoutEnlargement: true });
	pipeline = fmt === 'webp' ? pipeline.webp({ quality: 82 }) : pipeline.jpeg({ quality: 82, mozjpeg: true });
	await pipeline.toFile(cachePath);
	return cachePath;
}

export function invalidateCache(storedPath: string): void {
	for (const v of Object.keys(VARIANTS)) {
		for (const ext of ['jpg', 'webp']) {
			const p = `${storedPath}.${v}.${ext}`;
			if (fs.existsSync(p)) fs.unlinkSync(p);
		}
	}
}
```

## Endpoint variant

```ts
// routes/files.ts (ajout)
idaeApi.router.addRoute({
	method: 'get',
	path:   '/api/files/:fileId/image/:variant',
	handler: async (req, res) => {
		const meta = await FileService.getFileMeta(req.org, req.params.fileId);
		if (!meta || !meta.image) return res.status(404).end();
		const fmt = req.query.fmt === 'webp' ? 'webp' : 'jpeg';
		const variantPath = await ImageService.getVariant(meta._storedAbsolute, req.params.variant as ImageVariant, fmt);
		res.type(fmt === 'webp' ? 'image/webp' : 'image/jpeg');
		fs.createReadStream(variantPath).pipe(res);
	},
	middleware: [requireFileAccess('R')],
});
```

## Security

- Bombe ZIP / image bomb : `sharp` a un cap par défaut (`limitInputPixels: 268402689`). Garder le défaut, log si refusé.
- `variant` paramètre : whitelist exclusive (`'thumb'|'small'|'medium'|'large'`). Tout autre → 400.
- Pas d'EXIF strip ce sprint (sprint séparé sécurité si besoin).

## Acceptance criteria

1. Upload `image/jpeg` 1920×1080 → metadata enrichie `width:1920, height:1080, format:'jpeg', hasAlpha:false`
2. Upload `application/pdf` → metadata.image absent
3. GET `/api/files/:id/image/thumb` → 150×150 JPEG retourné
4. 2e GET même variant → servi depuis cache (vérifier mtime fichier cache inchangé)
5. GET variant `xxx` inconnu → 400
6. GET `?fmt=webp` → Content-Type `image/webp`
7. DELETE permanent → fichier original + tous variants cache supprimés
8. Image PNG avec alpha → resize garde alpha (output PNG ou WebP si demandé)
9. Image corrompue → upload OK (FileService) mais `metadata.image` reste absent (probe retourne null)
10. Suite no regression
