# S27-03 — MachineScheme.views getter

**Sprint:** 27
**Effort:** S
**Priority:** medium
**Dependencies:** S27-01

---

## Goal

Exposer `_views` de façon typée via MachineScheme pour que Explorer et autres composants n'aient pas à caster `any`.

## File

| File | Change |
|------|--------|
| `src/lib/main/machine/MachineScheme.ts` | Ajouter getter `views` |

## Implementation

```ts
import type { EntityViews } from '$lib/types/schema-types.js';

get views(): Partial<EntityViews> | undefined {
  return (this.model[this.collectionName] as any)?._views;
}
```

## Acceptance

- `machine.logic.collection('vehicle').views` retourne `Partial<EntityViews> | undefined`
- Typé — pas de cast `any` nécessaire dans les consommateurs
- Si `_views` absent → `undefined` (pas `{}`)
