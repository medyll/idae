## Edge cases

| Case | Expected |
|------|---------|
| Fresh install (no IDB) | No drift check needed — IDB created by createQoolie |
| Same schema, hash match | skip, no open/close overhead |
| New collection added | store created, others untouched |
| Collection deleted from model | store deleted, data gone (server sync reloads if needed) |
| Collection renamed | old deleted + new created (= data gone — server sync reloads) |
| `__outbox__` in IDB | never deleted, always protected |
| IDB open by another tab | upgrade blocked — catch + warn, proceed without upgrade |
