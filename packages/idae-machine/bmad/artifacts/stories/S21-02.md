## 1. Notation `free-*`

Regex strict :
```
^free-(?:(\d+)x(\d+)|w(\d+)|h(\d+))$
```

| Pattern | Width | Height | Default fit |
|---------|-------|--------|-------------|
| `free-125x88`  | 125 | 88  | `cover` (les 2 dims = intention crop) |
| `free-w800`    | 800 | —   | `inside` (hauteur libre, no enlarge) |
| `free-h420`    | —   | 420 | `inside` |

`free-` invalide (regex no match) → 400 + pas d'insertion DB.

## 2. Resolver

```ts
// server/src/services/ImageResolver.ts
import * as Registry from './ImagePresetRegistry.js';
import type { ImagePreset } from './ImagePresetRegistry.js';
import { config } from '../config.js';

const FREE_RE = /^free-(?:(\d+)x(\d+)|w(\d+)|h(\d+))$/;

export class PresetError extends Error {
	constructor(message: string, public status: 400 | 404) { super(message); }
}

export async function resolve(code: string): Promise<ImagePreset> {
	if (!code || code.length > 64) throw new PresetError('Invalid preset code', 400);

	// 1. Lookup DB
	const existing = await Registry.get(code);
	if (existing) return existing;

	// 2. Free notation
	const m = FREE_RE.exec(code);
	if (!m) throw new PresetError(`Unknown preset: ${code}`, 404);

	const [, wxh1, wxh2, wOnly, hOnly] = m;
	let width: number | undefined;
	let height: number | undefined;
	let fit: ImagePreset['fit'];

	if (wxh1 && wxh2) {
		width  = Number(wxh1);
		height = Number(wxh2);
		fit    = 'cover';
	} else if (wOnly) {
		width  = Number(wOnly);
		fit    = 'inside';
	} else if (hOnly) {
		height = Number(hOnly);
		fit    = 'inside';
	} else {
		throw new PresetError(`Invalid free notation: ${code}`, 400);
	}

	// 3. DOS bounds
	const { minDim, maxDim } = config.image;
	for (const dim of [width, height]) {
		if (dim === undefined) continue;
		if (dim < minDim || dim > maxDim) {
			throw new PresetError(`Dimension out of bounds [${minDim}..${maxDim}]: ${dim}`, 400);
		}
	}

	// 4. Insert auto preset
	const preset: ImagePreset = {
		code,
		name:    `Auto ${code}`,
		width,
		height,
		fit,
		format:  'auto',
		quality: 82,
		auto:    true,
		scope:   'global',
	};
	await Registry.insert(preset);
	return preset;
}
```

**Note** : `Registry.insert` met aussi à jour le cache mémoire, donc 2e appel concurrent → cache hit. Race condition possible sur 2 requêtes simultanées sur le même `free-WxH` jamais vu → MongoDB peut insérer 2× (pas d'unique index). Mitigation : créer index unique `{ code: 1 }` sur `appimage_preset` au boot (cf. S21-01 ajustement). À ajouter ici comme TODO si pas fait en S21-01.

→ **Action S21-02** : créer index unique au boot dans `seedImagePresets.ts` :
```ts
await coll.createIndex({ code: 1 }, { unique: true });
```
Puis dans `resolve` : `try { insert } catch (duplicate) { return Registry.get(code) }`.

## 4. Endpoint réécrit

```ts
// server/src/routes/files.ts (handler image variant)
idaeApi.router.addRoute({
	method: 'get',
	path:   '/api/files/:fileId/image/:preset',
	handler: async (req, res) => {
		try {
			const preset = await ImageResolver.resolve(req.params.preset);
			const meta = await FileService.getFileMeta(req.org, req.params.fileId);
			if (!meta || !meta.image) {
				return res.status(404).json({ error: 'Image not found' });
			}
			const fmt = req.query.fmt === 'webp' ? 'webp' : undefined;
			const variantPath = await ImageService.getVariant(
				path.join(config.filesRoot, req.org, meta.storedPath),
				preset,
				fmt
			);
			res.type(variantPath.endsWith('.webp') ? 'image/webp' : 'image/jpeg');
			fs.createReadStream(variantPath).pipe(res);
		} catch (err) {
			if (err instanceof ImageResolver.PresetError) {
				return res.status(err.status).json({ error: err.message });
			}
			logger.error('Image variant error:', err);
			res.status(500).json({ error: 'Image variant failed' });
		}
	},
	middleware: [requireFileAccess('R')],
});
```

(Note : `ImageService.getVariant` signature change → S21-03 refactor)

## Acceptance criteria

1. `GET /image/thumb` → preset DB trouvé, image servie
2. `GET /image/free-125x88` (jamais vu) → row inséré en DB avec `auto: true, fit: 'cover', w: 125, h: 88`, image servie
3. 2e `GET /image/free-125x88` → cache hit, pas de nouvelle insertion
4. `GET /image/free-w800` → row avec `width: 800, height: undefined, fit: 'inside'`
5. `GET /image/free-h420` → row avec `height: 420, fit: 'inside'`
6. `GET /image/free-9999x9999` → 400 `Dimension out of bounds`, **aucune insertion DB**
7. `GET /image/free-1x1` → 400 (sous `minDim`)
8. `GET /image/inexistant` (pas pattern free) → 404 `Unknown preset`
9. `GET /image/free-abc` (regex no match) → 404
10. `GET /image/free-w800?fmt=webp` → Content-Type `image/webp`
11. `GET /image/thumb` sur fichier non-image (mimetype pdf) → 404 `Image not found`
12. Concurrent `GET /image/free-200x100` ×10 simultané → un seul row inséré (index unique)
13. Suite no regression
