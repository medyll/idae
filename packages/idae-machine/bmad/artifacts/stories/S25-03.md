## Goal

Remplacer les 4 entrées fragmentées par une seule entrée `'explorer'`.
Vérifier que Frame.svelte passe correctement `vars.mode` comme prop `mode` à Explorer.

| File | Change |
|------|--------|
| `src/lib/main/router/componentRegistry.ts` | Retirer 4 entrées, ajouter 'explorer' |

## Changement registry

```ts
// SUPPRIMER
registry.register('explorer.list',    () => import('../shell/explorer/ExplorerList.svelte'))
registry.register('explorer.table',   () => import('../shell/explorer/ExplorerTable.svelte'))

// AJOUTER

```

## Frame.svelte prop passing

Frame reçoit `{ modulePath, collection, collectionId, vars }`. Il doit passer à Explorer:
- `collection` → prop `collection`
- `collectionId` → prop `collectionId`
- `vars?.mode` → prop `mode` (string cast vers union type)

```svelte
<!-- Dans Frame.svelte, au moment du mount dynamique -->
<Component
  collection={frame.collection}
  collectionId={frame.collectionId}
  mode={frame.vars?.mode ?? 'list'}
  vars={frame.vars}
/>
```

## Acceptance

- `machine.loadFrame('explorer', 'vehicle', undefined, { mode: 'list' })` → Explorer(collection='vehicle', mode='list')
- `machine.loadFrame('explorer', 'vehicle', '42', { mode: 'card' })` → Explorer(collection='vehicle', collectionId='42', mode='card')
- Ancien `loadIn('explorer.list', ...)` — si encore utilisé quelque part → mettre à jour vers `loadFrame`
- `pnpm run check` → 0 new errors
