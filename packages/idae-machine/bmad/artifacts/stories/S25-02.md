# S25-02 — Create Explorer.svelte — mode=list|table|card|actions

**Sprint:** 25
**Effort:** L
**Priority:** high
**Dependencies:** S25-01, S24-04 (Frame), S24-03 (machine.loadFrame)

---

## Context

Quatre composants séparés (ExplorerList, ExplorerTable, ExplorerCard, ExplorerActions) dans componentRegistry. Logique dupliquée, props inconsistants, navigation via `machine.loadIn()` (deprecated dans ce contexte).

## Goal

Un seul `Explorer.svelte` avec prop `mode` qui dispatche l'affichage. Navigation via `machine.loadFrame()` uniquement.

## Files

| File | Change |
|------|--------|
| `src/lib/shell/explorer/Explorer.svelte` | CREATE |

## Props

```ts
let {
  collection,
  mode = 'list',
  collectionId,
  vars,
}: {
  collection:    string;
  mode?:         'list' | 'table' | 'card' | 'actions';
  collectionId?: string;
  vars?:         Record<string, string>;
} = $props();
```

## Dispatch par mode

```svelte
{#if mode === 'list'}
  <DataList {collection} sortBy={scheme?.defaultSort}>
    {#snippet children({ items, index })}
      {#each items as row (row[index])}
        <div class="list-row" onclick={() => machine.loadFrame('explorer', collection, String(row[index]), { mode: 'card' })}>
          <!-- presentation fields -->
        </div>
      {/each}
    {/snippet}
  </DataList>

{:else if mode === 'table'}
  <DataList {collection} sortBy={scheme?.defaultSort}>
    {#snippet children({ items, index })}
      <table><!-- headers + rows --></table>
    {/snippet}
  </DataList>

{:else if mode === 'card'}
  <!-- load single record -->
  <CardForm {collection} id={collectionId} mode="update" />

{:else if mode === 'actions'}
  <DataList {collection}>
    {#snippet children({ items, index })}
      {#each items as row (row[index])}
        <button onclick={() => machine.loadFrame('explorer', collection, String(row[index]), { mode: 'card' })}>
          <!-- label -->
        </button>
      {/each}
    {/snippet}
  </DataList>
{/if}
```

## Acceptance

- Record click (list/table/actions) → `machine.loadFrame('explorer', collection, id, { mode: 'card' })`
- mode='card' + collectionId → CardForm charge le record
- mode='card' sans collectionId → CardForm mode='create'
- Svelte 5 runes uniquement
- Fallback gracieux si collection inconnue
- `template.presentation` utilisé pour colonnes (S27 câblera `_views`)
