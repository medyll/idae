## User Story

As a **library internals developer**, I want a standalone `cache.ts` module so that **the router can serve stale data immediately and revalidate in the background**, eliminating perceived fetch latency on back-navigation without introducing any external dependency.

---

## Context

Cache is an in-memory `Map`. No `localStorage` persistence. Strategy is stale-while-revalidate (SWR). The cache is consumed exclusively by `router.ts` (S04-04). This module is tree-shakeable: if `cache: false` globally, the import of `cache.ts` is never reached.

---

## Acceptance Criteria

```gherkin
Given a route fetch that resolves with data for URL '/api/users/42'
When the result is stored via CacheStore.set
Then getCacheEntry returns the entry (cache hit)

Given an entry stored more than ttl ms ago
When CacheStore.get is called for that key
Then undefined is returned (TTL expired, entry evicted)

Given an entry stored less than staleTime ms ago
When CacheStore.get is called
Then state 'fresh' is returned (no revalidation needed)

Given an entry stored between staleTime and ttl ms ago
When CacheStore.get is called
Then state 'stale' is returned (serve + revalidate)

Given createCacheStore(false) (disabled)
When set/get/invalidate are called
Then all operations are no-ops (nothing stored or returned)

Given invalidate('/users/*') is called
When get is called for '/api/users/42'
Then undefined is returned (entry cleared by glob)

Given deepEqual({ a: 1 }, { a: 1 })
Then true is returned

Given deepEqual({ a: 1 }, { a: 2 })
Then false is returned
```

**New file:** `src/lib/cache.ts`

```typescript
export interface CacheEntry {
  data: unknown;
  timestamp: number;
  url: string;
  status: number;
}

export interface CacheStore {
  get(key: string, ttl?: number, staleTime?: number):
    { entry: CacheEntry; state: 'fresh' | 'stale' } | undefined;
  set(key: string, entry: CacheEntry): void;
  invalidate(pattern: string): void;
  clear(): void;
}

/**
 * Create an in-memory SWR cache store.

 * @returns CacheStore instance.
 * @public
 */
export function createCacheStore(enabled: boolean): CacheStore { ... }

/**
 * Deep-equal comparison (recursive). Used to guard action re-invocation.
 * @public
 */
export function deepEqual(a: unknown, b: unknown): boolean { ... }
```

Cache key formula: `"${method.toUpperCase()} ${resolvedUrl}"` (e.g. `"GET /api/users/42"`)

Glob matching for `invalidate` — no external library:
- Exact key: clear that one entry only.
- Pattern ending in `/*` or `*`: use `key.startsWith(prefix)` where prefix is the pattern without `*`.

Default TTL: `60_000` ms. Default staleTime: `0` ms.
