# AGENTS.md

## Agents in the `idae-slotui` project

This document describes the agents (human or automated) involved in the development, maintenance, and automation of the `@medyll/idae-slotui-svelte` project.

### 1. Human Agents

- **Lead Developers:**
  - @medyll (main maintainer)
  - Contributors listed in the main repository
- **Reviewers:**
  - Members of the `idae` team

### 2. Automated Agents

- **GitHub Copilot:**
  - Used for code generation assistance, documentation, and automation of repetitive tasks.
- **CI/CD (GitHub Actions):**
  - Build, test, lint, release (see `.github/workflows/` in the monorepo)
- **Dependency Bots:**
  - Dependabot (dependency updates)

### 3. Agent Roles

- **Development:** Feature addition, bug fixing, refactoring.
- **Documentation:** Generation and update of documentation (README, automated docs, etc.).
- **Testing:** Running unit and integration tests.
- **Release:** Publishing packages and generating CSS/JS artifacts.

### 4. Best Practices

- Document any automation or agent added in this file.
- Keep the list of agents and their roles up to date.
- Specify critical automated workflows.
