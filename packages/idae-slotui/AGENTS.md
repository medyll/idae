## Migration SCSS → Tailwind (clôturée)

Tous les composants du package idae-slotui sont désormais migrés vers Tailwind CSS (avec UnoCSS et tokens CSS personnalisés).

- Les nouveaux composants doivent utiliser Tailwind/UnoCSS et consommer les tokens CSS générés (slotui-css).
- Les fichiers SCSS restants servent uniquement à la génération des tokens CSS et à la compatibilité thèmes.
- Voir MIGRATION_SCSS_TAILWIND.md pour l’historique et les détails.

_Dernière mise à jour : 2026-01-18_
