# AppBar

Barre d’application supérieure, équivalent Material-UI.

## Props
- position: 'fixed' | 'absolute' | 'sticky' | 'static' | 'relative' (défaut: 'fixed')
- color: 'primary' | 'secondary' | 'default' | 'transparent' (défaut: 'primary')
- class, style

## Utilisation
```svelte
<AppBar position="sticky" color="secondary">
  <Toolbar>Mon App</Toolbar>
</AppBar>
```
