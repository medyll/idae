# Composants de base (base)

Ce dossier contient les composants atomiques de l’UI (ex : Alert, Badge, Box, etc.).

Chaque composant doit avoir sa documentation JSDoc dans le <script context="module"> du .svelte, et un README.md local si besoin de détails d’usage ou d’exemples.

- Pour la structure des props, voir le fichier TypeScript associé ou la doc JSDoc dans le composant.
- Pour les styles, voir le SCSS/Tailwind associé.

- Avatar

- Badge
