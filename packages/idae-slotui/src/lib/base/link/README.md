# Link

Lien stylisé, équivalent Material-UI.

## Props
- href: string (défaut: '#')
- target: string (défaut: '_self')
- rel: string
- underline: 'none' | 'hover' | 'always' (défaut: 'hover')
- color: string (défaut: 'primary')
- class, style

## Utilisation
```svelte
<Link href="/docs" underline="always">Documentation</Link>
```
