# Typography

Composant de texte, équivalent Material-UI.

## Props
- variant: 'h1'...'h6' | 'subtitle1' | 'subtitle2' | 'body1' | 'body2' | 'caption' | 'overline' (défaut: 'body1')
- color: string (défaut: 'inherit')
- align: 'left' | 'center' | 'right' | 'justify' (défaut: 'left')
- gutterBottom: boolean
- noWrap: boolean
- class, style

## Utilisation
```svelte
<Typography variant="h2" color="primary" align="center">Titre</Typography>
```
