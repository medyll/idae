# Fab (Floating Action Button)

Bouton d’action flottant, équivalent Material-UI.

## Props
- icon: string (optionnel)
- color: string (défaut: 'primary')
- size: 'small' | 'medium' | 'large' (défaut: 'medium')
- class, style, ariaLabel

## Utilisation
```svelte
<Fab icon="mdi:add" color="secondary" />
```
