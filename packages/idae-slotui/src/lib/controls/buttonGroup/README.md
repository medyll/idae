# ButtonGroup

Groupe de boutons alignés, équivalent Material-UI.

## Props
- orientation: 'horizontal' | 'vertical' (défaut: 'horizontal')
- variant: 'text' | 'outlined' | 'contained' (défaut: 'text')
- size: 'small' | 'medium' | 'large' (défaut: 'medium')
- class, style

## Utilisation
```svelte
<ButtonGroup orientation="vertical">
  <Button>Un</Button>
  <Button>Deux</Button>
</ButtonGroup>
```
