# Controls Components (controls)

This folder contains interactive/form components (e.g., Button, Checkbox, Select, etc.).

- Each component must have its JSDoc documentation in the <script lang="ts"> block of the .svelte file.
- Add usage examples or specific notes here if needed.

## List of components

- Autocomplete
- Button
- Checkbox
- Confirm
- InPlaceEdit
- Progress
- Rating
- Select
- Slider
- Stepper
- Switch
- TextField
