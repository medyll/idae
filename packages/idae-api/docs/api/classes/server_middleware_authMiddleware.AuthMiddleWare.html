    Class AuthMiddleWare

    AuthMiddleWare

    Fournit la gestion JWT pour l'authentification, la génération, la vérification et le rafraîchissement de tokens. Expose un middleware Express pour la vérification des tokens et des routes pour login/logout/refresh.

    Example

    const auth = new AuthMiddleWare(secret, expiration)
      app.use(auth.createMiddleware())
      auth.configureAuthRoutes(app)
