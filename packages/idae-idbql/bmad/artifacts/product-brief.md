# Product Brief – @medyll/idae-idbql

## Overview
A powerful and flexible IndexedDB query library for TypeScript and JavaScript applications, offering a MongoDB-like query interface, strong TypeScript support, reactive state management, and easy integration with front-end frameworks like Svelte.

## Goals
- Provide a MongoDB-like query interface for IndexedDB
- Enable reactive state management for frontend apps
- Support Svelte 5 Runes and idae-stator

## Target Users
- Svelte developers
- TypeScript/JavaScript app builders
- Anyone needing browser-side persistence

## Success Metrics
- Adoption in Svelte projects
- Community feedback
- Test coverage
