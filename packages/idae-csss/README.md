# idae-csss

CSS and Style utility library for the Idae ecosystem.

## Architecture

```mermaid
graph TD
    A[CSS Rules] --> B[Style Utilities]
    B --> C[Class Names]
    B --> D[Theme Management]
    C --> E[DOM Application]
    D --> E
```

## Features

- Style utilities
- Class composition
- Theme management
- Responsive design

## Installation

```bash
npm install @medyll/idae-csss
pnpm add @medyll/idae-csss
```
