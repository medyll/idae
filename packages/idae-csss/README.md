# @medyll/idae-csss

`idae-csss` is a utility library for CSS manipulation and type-safe style generation within the Idae ecosystem.

## Features (Planned)

- **Type-safe CSS Properties**: Define and use CSS properties with TypeScript validation.
- **Fluent API**: Generate styles using a chainable, intuitive API.
- **PostCSS Integration**: Plugin-ready architecture for advanced CSS transformations.
- **Dynamic Type Generation**: Automatically generate TypeScript interfaces from CSS models.

## Installation

```bash
pnpm add @medyll/idae-csss
```
